Code for Kustau

Code for Kusatuは、私の住むりっとうのお隣、草津市をメインに活動するCode for ブリゲード、シビックテック団体です。 議員、保育士、その他行政職員、一般企業職、無職や中学生、幼児など、多様な人々が参加しています。私は、コードがかけるので、技術担当として入っています。 本当にここの皆様には学ぶことも多く、普通に高校生していては絶対に知り得ない知見もたくさんいただいていて、感謝してもし尽くせません。 私に5374.jpの存在を教えてくれたのもC4Kでした。本当に感謝しています。
これまでに、私が最もコミットしたのは、チャレンジ・オープンガバナンス2018に応募した2つのプロジェクトです。 一方は、最高賞たるオープンガバナンス総合賞を獲得しました。このプロジェクトには、技術的な側面から意見を述べたり、 システムの大まかな構成を練ったほか、提出,プレゼン資料に用いるプロトタイプなどの作成にあたりました。 写真は、COG2018最終審査プレゼン時のものです。
